### 对话逻辑

通过唤醒词or人体感应触发，即进入反馈语播报，同时开始计时，计时内，不记拾音次数；

1. 唤醒时，随机播放唤醒反馈语；

2. 获取**有效**文本时，返回拾音结果后，重新进入拾音，重置T，S；

3. 获取**无效**文本时，随机播报引导语，T+1，重置S；

4. 结束时，随机播放结束语；

补充

- 拾音计时时长：10s **（暂定）**

- 结束后保留在当前界面，60s后返回待机页面。

| 名词 | 描述             |
| ---- | ---------------- |
| T    | 引导次数         |
| S    | 时间，单位：s/秒 |
